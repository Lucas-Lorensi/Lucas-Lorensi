### Featured AI projects

- **rag-rbac-server** — A RAG API where retrieval is filtered by user permissions: a user's assistant can only surface documents they're authorized to see. Access control enforced *at the retrieval layer*, not after generation. &nbsp;`FastAPI` · `Azure AI Search`
- **sql-agent** — Natural-language questions over a database, answered with conversational insights. The query step is isolated as its own validated agent. &nbsp;`LangGraph` · `SQL`
- **llm-guardrails** — Guardrail patterns for LLM apps: prompt-injection filtering, output validation, and tool-scope constraints.
